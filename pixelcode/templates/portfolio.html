Our Work

PHP

PHP has a wide variety of applications on the Web, may it be a simple website or a facebook app. It can be a fast and powerful tool to add dynamic content on your site. It can also be used in combination with a database backend such as MySQL, Sqlite etc.

Python

Python is a programming language that focuses on high-level operations. It lets us work more quickly and integrate systems more effectively. It is recommended for websites and applications that require complex application logic and background tasks. We utilize various frameworks built in python that help create faster and more robust sites and applications with dynamic user generated content.

Java

Java programs used within a web page can add many new features to a web browser. These are small program used in the programming of instant messaging, chat service, solving some complex calculation and for many other purposes. It is also used to create new and advanced cross platform applications.

MySQL

MySQL is a popular choice of database for use in web applications and is a common component of our LAMP Stack. We use it conjunction with many of our sites and web apps that require a well tested and proven database backend.

CouchDB

A growing number of applications are moving away from the traditional Relational database solutions such as MySQL to highly scalable and fast ones such as CouchDB. If you require a web application that needs to perform hundreds of queries in seconds, CouchDB may be the right choice for you. It can be used in conjunction with PHP/Python/Rails to make your web app complete.

Code Igniter

This is a small, simple and fast PHP Framework that can be used to build small web applications to larger cross platform web softwares. It's clean code and straight forward design makes it a popular php framwork at PixelCode.

PostgreSQL

This is a popular alternative to MySQL as a database engine. It boasts of greater security, optimization, error handling and lesser quirks and limitations than its competitor. If MySQL doesn't quite fit the very specific requirements of your project, PostgreSQL might be the right tool for you.

MongoDB

MongoDB is a popular alternative to CouchDB. They are both document stores that allows you to perform lighting fast, deep queries. It is also schema free allowing your database to take whatever shape it needs to depending on your code. This provides your web application a clear path to horizontal scalability.
